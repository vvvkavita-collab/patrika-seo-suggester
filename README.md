# Patrika SEO Suggester

Paste any news article and get Google/SEO-ready suggestions:
- Suggested Title & Meta (as per Google SEO guideline)
- Keywords, URL Slug, NewsArticle JSON-LD
- Internal links, Image alt text, Readability fixes
- Downloads: DOCX, JSON-LD, HTML, CSV

## Run locally
pip install -r requirements.txt
streamlit run app.py

## Deploy
Push to GitHub, deploy on Streamlit Cloud with app.py as entrypoint.
